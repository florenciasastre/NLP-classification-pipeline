# Clasificación de Texto con NLP – Pipeline Completo

Proyecto académico – Máster en Inteligencia Artificial (EDEM)

## Objetivo

Desarrollar un pipeline completo de Procesamiento de Lenguaje Natural (NLP) para la clasificación binaria de reseñas utilizando un dataset real.

## Pipeline

- Análisis exploratorio de los datos
- Limpieza y normalización del texto
- División del dataset (Train / Validation / Test – 70/15/15)
- Modelo base: TF-IDF + Regresión Logística
- Modelo neuronal: LSTM bidireccional
- Evaluación con F1-score, matriz de confusión y curva ROC

## Tecnologías utilizadas

Python · Pandas · Scikit-learn · TensorFlow / Keras · Matplotlib

## Contenido del repositorio

El notebook incluye la implementación completa del pipeline, el entrenamiento de los modelos y la comparación de resultados.
